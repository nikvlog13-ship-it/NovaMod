# NovaMod

Версия: Beta 1.1.0

Используйте F5 для того, чтобы открыть чек-лист.

Используйте F6 для того, чтобы сменить язык, доступные языки:
* Русский;
* Английский

## Установка

Установите .dll и .pdb файлы в папку ``Hollow Knight_Data/Managed/Mods/NovaMod``. Не забудьте установить Lumafly.
